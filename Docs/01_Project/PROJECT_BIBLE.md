# PrisonGame — Project Bible

## 1. Project definition

**Genre:** cooperative prison-life simulation with escape possibilities.

**Core idea:** escape is the most ambitious shared project, but it is not the only reason to play. Daily prison life must be entertaining through professions, relationships, economy, cell development, time pressure, underground activities and player-created stories.

**Target playthrough length:** approximately 5–10 hours.

## 2. Desired player stories

Players should remember moments such as:

- passing a tool through a vent before a search;
- one player finishing an order while another distracts a guard;
- specializing in different professions and becoming useful to each other;
- hiding contraband in a friend's cell;
- missing a shop window because a fight or NPC request changed the plan;
- coordinating a nighttime escape step while the prison block remains active.

## 3. Design pillars

### Every day creates a different story

The schedule is understandable, but events and opportunities change.

### Cooperation emerges naturally

Players naturally specialize and become useful through skills, access, trust, materials and knowledge.

### Work is gameplay

Work supports official orders, underground orders, theft, cooperation, fights, trust and access.

### The cell is a personal base

The cell visibly reflects progression through posters, lighting, storage, hidden stashes, radio, trophies and profession equipment.

### Professions follow the player home

At medium trust, professions unlock small cell workstations without replacing full workplaces.

### One shared residential block

Players and NPCs see each other through bars at night. Night remains an active gameplay phase.

## 4. Economy and progression

Important resources:

1. official currency or coupons;
2. cigarettes and valuables;
3. trust and reputation;
4. time;
5. materials.

## 5. Escape

The first full escape route is based on ventilation.

High-level flow:

1. obtain a usable tool;
2. remove bolts;
3. open the vent;
4. pass technical obstacles;
5. obtain a specialized tool;
6. use professions, NPCs and materials;
7. complete the final cooperative stage.

## 6. Scope philosophy

Before adding many professions, buildings or escape routes, finish:

- one full playable day;
- one meaningful profession;
- one complete container loop;
- one complete escape route;
- one cooperative gameplay loop.
